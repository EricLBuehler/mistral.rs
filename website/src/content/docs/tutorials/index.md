Linear step-by-step lessons.

## Available tutorials

1. [Install and run your first model](/mistral.rs/tutorials/01-install-and-run/): install the binary and chat with a model in the terminal.
2. [Serve a model as an API](/mistral.rs/tutorials/02-serve-an-api/): expose a model over an OpenAI-compatible HTTP endpoint.
3. [Use the Python SDK](/mistral.rs/tutorials/03-python-sdk/): call models from Python without a separate server.
4. [Use the Rust SDK](/mistral.rs/tutorials/04-rust-sdk/): embed mistral.rs in a Rust application.
5. [Build an agent](/mistral.rs/tutorials/05-build-an-agent/): enable tool calling, web search, and code execution.
6. [Quantize a model](/mistral.rs/tutorials/06-quantize-a-model/): fit a larger model on a smaller GPU.

Later tutorials assume earlier ones are complete.
