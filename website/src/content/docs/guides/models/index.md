title: Work with specific model types
description: Vision, image generation, speech, and embedding workloads. What each one expects and how to call them.
---

mistral.rs handles a lot more than text chat. Each model category has its own request shape and its own set of things worth knowing. These guides cover the ones you are most likely to reach for.

- [Use vision and video input](/mistral.rs/guides/models/use-vision-input/): image and video handling across models that accept them. Qwen3-VL and Gemma 4 are the main examples.
- [Use image generation models](/mistral.rs/guides/models/use-image-generation/): running diffusion models like FLUX.
- [Use speech models](/mistral.rs/guides/models/use-speech-models/): Voxtral for speech-to-text, Dia for text-to-speech.
- [Use embedding models](/mistral.rs/guides/models/use-embeddings/): EmbeddingGemma, Qwen3-Embedding, and what to do with the vectors.

For the full list of models we support and what each accepts, see the [supported models reference](/mistral.rs/reference/supported-models/).
