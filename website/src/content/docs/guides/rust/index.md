[Tutorial 4](/mistral.rs/tutorials/04-rust-sdk/) covers the basics of the Rust SDK. These guides pick up from there.

- [Streaming](/mistral.rs/guides/rust/streaming/): working with futures streams from the chat API.
- [Embed in Axum](/mistral.rs/guides/rust/embed-in-axum/): mount mistralrs as a service inside an existing Rust web app.

The API surface lives in [Rust reference](/mistral.rs/reference/rust-api/). Crate-level docs are on [docs.rs](https://docs.rs/mistralrs).
