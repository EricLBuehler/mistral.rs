[Tutorial 3](/mistral.rs/tutorials/03-python-sdk/) covers the basics of the Python SDK. These guides pick up from there and cover the parts you are most likely to reach for next.

- [Streaming](/mistral.rs/guides/python/streaming/): iterating over tokens as they generate.
- [Multimodal input](/mistral.rs/guides/python/multimodal-input/): images, audio, and video from Python.
- [Agentic sessions from Python](/mistral.rs/guides/python/agentic-session/): persistent state across multiple calls.

The full API surface lives in the [Python reference](/mistral.rs/reference/python-api/).
