Enum NextByte

pub enum NextByte {
    ForcedByte(u8),
    ForcedEOI,
    SomeBytes0,
    SomeBytes1(u8),
    SomeBytes2([u8; 2]),
    Dead,
}

Variants§

§

ForcedByte(u8)

Transition via any other byte, or EOI leads to a dead state.

§

ForcedEOI

Transition via any byte leads to a dead state but EOI is possible.

§

SomeBytes0

Transition via some bytes may be possible. The bytes are possible examples.

§

SomeBytes1(u8)

§

SomeBytes2([u8; 2])

§

Dead

The current state is dead. Should be only true for NO_MATCH.
