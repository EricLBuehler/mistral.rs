pub type ChatCompletionOnChunkCallback = OnChunkCallback<ChatCompletionChunkResponse>;
Expand description

A callback function that processes streaming response chunks before they are sent to the client.

This hook allows modification of each chunk in the streaming response, enabling features like content filtering, transformation, or logging. The callback receives a chunk and must return a (potentially modified) chunk.

    chunk
