# Domain Adapter: Product

Assessment and evolution knowledge for product design and development.

## Assessment Criteria

### User Experience
- **Usability**: Task completion rates, error rates, time-on-task
- **Accessibility**: WCAG compliance level (A, AA, AAA)
- **Responsiveness**: Fast load times, smooth interactions
- **Information architecture**: Logical navigation, findability
- **Visual design**: Consistency, modern aesthetics, brand alignment

### Feature Coverage
- **Core features**: Are all essential features implemented?
- **User journeys**: Are key user flows complete and optimized?
- **Edge cases**: Are error states, empty states handled?
- **Platform coverage**: Desktop, mobile, tablet support

### Design System
- **Component library**: Coverage, consistency, documentation
- **Design tokens**: Color, typography, spacing defined
- **Pattern library**: Common interaction patterns documented
- **Accessibility patterns**: Focus management, screen reader support

### User Satisfaction
- **NPS score**: Net Promoter Score
- **Support tickets**: Volume and categories
- **Feature requests**: Common themes
- **User reviews**: Sentiment analysis

## Analysis Criteria

### Competitive Landscape
- Search for: `"{product_category} best {year}"`
- Search for: `"{competitor_product} features comparison"`
- Search for: `"UX trends {product_category} {year}"`

### Design Trends
- UI/UX design trends in the category
- Accessibility standards updates
- Platform design guideline changes (iOS HIG, Material Design)
- New interaction paradigms

## Planning Patterns

### Quick Wins
- Fix accessibility issues
- Improve error messages
- Optimize load times
- Polish visual inconsistencies

### Strategic Improvements
- Redesign key user flows
- Build design system
- Add platform support (mobile, tablet)
- Implement user research program

## Health Indicator Thresholds

| Indicator | Healthy | Warning | Critical |
|---|---|---|---|
| Accessibility (WCAG) | AA compliant | Partial AA | No compliance |
| Load time | <2s | 2-5s | >5s |
| Feature completeness | >90% | 70-90% | <70% |
| Design consistency | >90% components | 70-90% | <70% |
| User satisfaction | >4.0/5 | 3.0-4.0/5 | <3.0/5 |
