---
title: Customize
description: Adapters, chat templates, and sampling behavior.
---

Control how a model loads and generates:

- [LoRA, X-LoRA, and AnyMoE](/mistral.rs/guides/customize/lora-adapters/) - attach fine-tuned adapters or compose experts at inference time.
- [Chat templates](/mistral.rs/guides/customize/chat-templates/) - override the prompt format when auto-detection is wrong.
- [Sampling parameters](/mistral.rs/guides/customize/sampling/) - temperature, top-k/top-p/min-p, penalties, and DRY.
