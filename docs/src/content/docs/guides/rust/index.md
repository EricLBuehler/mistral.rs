title: Rust SDK
description: Embed the mistral.rs engine into a Rust program.
---

The `mistralrs` crate embeds the engine into a Rust program.

- [Getting started](/mistral.rs/guides/rust/getting-started/) - load a model in-process and send a chat request.
- [Streaming](/mistral.rs/guides/rust/streaming/) - handle `Response` variants, tool progress, and cancellation.
- [Embed in Axum](/mistral.rs/guides/rust/embed-in-axum/) - mount the HTTP API inside an existing Axum app.
- [Rust SDK reference](/mistral.rs/reference/rust-sdk/) - summary of the `Model` API.
- [docs.rs](https://docs.rs/mistralrs) - full API documentation.
